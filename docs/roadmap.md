# Development Roadmap

This roadmap outlines the milestones and release goals for the ANCH Framework.

---

## 🟩 v0.1.0 — Foundation (Completed)
- [x] Python core library (`anch`).
- [x] Basic modules: features, neural, logistic chaos, permutation, and Feistel compression.
- [x] Command Line Interface (CLI) tools.
- [x] Pytest unit test coverage.
- [x] Initial PyPI package release (`anch-hash`).

---

## 🟨 v0.2.0 — Optimization & Integrations (Completed)
- [x] Fast optional **NumPy acceleration** for vectorized feature extraction.
- [x] High-performance **FastAPI REST API server** exposing all hashing features.
- [x] Dynamic **Next.js interactive Playground and Speed Test Dashboard**.
- [x] Stable attracting bounds for chaos map equations.

---

## 🟪 v0.3.0 — Enhanced Cryptographics (Completed)
- [x] Multi-chaotic map engine (integrated **Tent Map** and **Hénon Map**).
- [x] Adaptive map selection based on neural seed checksums (`seed % 3`).
- [x] Integration-ready structure for alternative parameter weights.

---

## 🟦 v1.0.0 — Production Release (Completed)
- [x] Full-coverage documentation site (MkDocs with Material theme).
- [x] GitHub Actions automated CI/CD pipeline configuration.
- [x] Stable API lock for extension plugins.
